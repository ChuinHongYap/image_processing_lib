# image_processing_lib

This is a repository of some useful image processing tools.

# Packages Used
OpenCV 4.0

Numpy 1.18

Matplotlib 3.2

Pillow 8.0.1

# Image Augmentation
Image augmentations performed are:
1. Grayscale
2. Flip horizontally and vertically
3. Change contrast
4. Image rotation
